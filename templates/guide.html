🎯 Giới Thiệu

Chào mừng bạn đến với Ứng Dụng Quản Lý Báo Cáo Tự Động - giải pháp toàn diện giúp tự động hóa quy trình tải và quản lý báo cáo của bạn.

🔍 Tính Năng Chính

  • Tự Động Hóa: Tự động tải báo cáo theo lịch trình
  • Đa Nền Tảng: Hỗ trợ nhiều nguồn dữ liệu khác nhau
  • Bảo Mật: Mã hóa dữ liệu an toàn
  • Dễ Sử Dụng: Giao diện trực quan, thân thiện
  • Tùy Chỉnh: Cấu hình linh hoạt theo nhu cầu

Dưới đây là các vấn đề thường gặp và cách khắc phục:

Cách khắc phục:

  1. Kiểm tra cổng 5000: ```bash # Trên Windows netstat -ano | findstr :5000

# Trên macOS/Linux lsof -i :5000 ``` - Nếu cổng đang được sử dụng, hãy đóng ứng dụng đang chiếm cổng hoặc đổi cổng trong cấu hình

  1. Kiểm tra file cấu hình:
  2. Đảm bảo file config.json tồn tại trong thư mục cấu hình

  3. Kiểm tra quyền đọc/ghi đối với file cấu hình

  4. Xem log lỗi:

  5. Kiểm tra file log trong thư mục logs/
  6. Tìm kiếm các thông báo lỗi gần nhất
